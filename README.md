# Test This Project

This project demonstrates a simple GUI for loading 2D points and exporting them
as DXF files.

## Requirements

The optional DXF export feature depends on [`ezdxf`](https://pypi.org/project/ezdxf/).
Install the requirements with:

```sh
pip install -r requirements.txt
```

## Usage

Run the GUI application:

```sh
python -m gui.app
```
